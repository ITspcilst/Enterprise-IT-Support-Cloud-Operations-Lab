# Enterprise IT Support & Cloud Operations Lab

> A realistic enterprise cloud environment built using Microsoft Azure, Microsoft Entra ID, Microsoft 365 and PowerShell.

---

## Executive Summary

This project simulates the daily responsibilities of an IT Support Engineer working within a modern hybrid organisation.

Rather than demonstrating isolated Azure services, this repository showcases how Microsoft Azure, Microsoft Entra ID and Microsoft 365 work together to support users, secure identities, manage cloud resources, automate administration and resolve real-world IT incidents.

The project follows enterprise best practices and is documented as if it were an internal IT implementation used within a production environment.

## Business Scenario

This project is based on a fictional UK financial services organisation migrating from an ageing on-premises environment to Microsoft Azure.

The IT department is responsible for supporting more than 300 hybrid employees while maintaining security, business continuity and operational efficiency.
