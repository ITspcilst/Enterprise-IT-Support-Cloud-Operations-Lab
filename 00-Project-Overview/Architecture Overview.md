# Architecture Overview

## Environment Overview

The organisation operates a hybrid Microsoft cloud environment.

Microsoft Entra ID provides identity services for all users.

Microsoft 365 delivers collaboration and productivity services.

Azure hosts cloud infrastructure including networking, virtual machines, storage and monitoring.

PowerShell automates administrative tasks.

---

## High-Level Components

- Microsoft Entra ID
- Microsoft 365
- Azure Subscription
- Resource Groups
- Azure Virtual Network
- Azure Virtual Machines
- Azure Storage
- Azure Monitor
- Recovery Services Vault
- Defender for Cloud

---

## Architecture Goals

- Secure identities
- High availability
- Centralised management
- Least privilege
- Automated administration
