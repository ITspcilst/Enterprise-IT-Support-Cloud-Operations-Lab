# Project Objectives

## Primary Objective

Build a realistic Microsoft cloud environment demonstrating the day-to-day responsibilities of an enterprise IT Support Engineer.

---

## Technical Objectives

- Deploy Azure resources
- Configure Microsoft Entra ID
- Implement Microsoft 365
- Configure RBAC
- Secure identities
- Deploy Azure Virtual Machines
- Configure Azure Storage
- Configure Monitoring
- Configure Backup
- Automate administration using PowerShell

---

## Documentation Objectives

Produce professional documentation suitable for GitHub and technical interviews.

---

## Career Objectives

Demonstrate practical skills for roles including:

- IT Support Technician
- Help Desk Engineer
- Microsoft 365 Administrator
- Azure Administrator
- Cloud Support Engineer
